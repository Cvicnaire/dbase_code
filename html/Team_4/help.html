Passage Form

For each field in the form: Name: input the name of the person filling the form. Genoype: select genotype of line passaged. BC/IB Number: input current inbreed or backcross number of the generation, adding +1 for the latest cross. Hatch Date: hatch date of the line passaged. Passage Date: input today's date. Mating Line: the the passage is crossing with.

Sorting Form

For each field in the form: Name: input the name of the person filling the form. Date Sorting: input today's date. Marker Location: input location of fluorescent marker. Ratio of fluorescent to non-fluorescent larvae: approximate ratio using slider, with color representing fluorescent mosquitos. Number of fluorescent animals sorted: input total number of animals which were fluorescent. Additional Notes: open field for any relevant notes, such as male to female ratios.

Clutch Form

For each field in the form: Name: input the name of the person filling the form. Genoype: select genotype of line passaged. Hatch Date: input hatch date of the clutch. Collection Date: input date the eggs are collected. Clutch number: is it the first, second, or third time collecting eggs from this cage since last passage? Egg Papers: input density of the eggs on the egg paper. See images below for reference.

New Line form

This form allows the user to add a new line into the databsase.

This provides as a supplementary page that with a table of all of the lab users registered into the website operations. The add new user button will allow the user to submite a new lab member into the database.

After entering the administrator password, select the table to edit from the dropdown menu. Enter the sort, passage, or clutch ID of the row you want to edit in the first field, and allow the form to autopopulate with the current values in the table. The form allows users to freeform edit any field in the row. After editing the desired values, click submit. A confirmation message will appear on screen if changes were successful.

Clutch Wellness

The clutch wellness functionality allows users to perform clutch wellness checks. Select the Line in question from the genotype dropdown menu. The graph will display the egg density of the line over time as pulled from record submitted through the clutch form. Each individual line is correlated with whether the clutch was the first, second, or third of its parent line.
